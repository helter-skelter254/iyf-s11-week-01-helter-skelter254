# Task 1.2: DevTools Exploration

## Website 1: https://example.com
1. **What HTML tags are used on the page?**
   * Answer: `<html>`, `<head>`, `<body>`, `<div>`, `<h1>`, `<p>`, and `<a>`.
2. **What is the page title?**
   * Answer: Example Domain
3. **How many headings are there?**
   * Answer: Exactly 1 heading (an `<h1>` tag reading "Example Domain").

## Website 2: https://developer.mozilla.org
1. **Find the navigation menu - what tag is it wrapped in?**
   * Answer: It is wrapped in a `<nav>` tag with the class `navigation`.
2. **How is the search bar structured?**
   * Answer: It uses an `<input>` tag with `type="search"`.
3. **What happens when you hover over links (check the styles)?**
   * Answer: The mouse cursor changes to a pointer, and the text color shifts via the CSS `:hover` pseudo-class.

## Website 3: Google.com
1. **Identify 5 different HTML elements:**
   * Answer: `<body>`, `<form>`, `<input>`, `<a>`, and `<button>`.
2. **Find a form element and list its inputs:**
   * Answer: The main search form contains a text input tag (`<input type="text">`) for typing queries and hidden configurations.
